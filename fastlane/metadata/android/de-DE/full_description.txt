<p>Acastus-Photon übersetzt den Namen einer Straße/Stadt/Sehenswürdigkeit in Geo Koordinaten, die</p>
<ul>
<li>in einer Navigations-/Karten App angezeigt werden können (z.b. OsmAnd, Maps.me, APhotoManager)</li>
<li>über die "Teilen"-Funktion von Android verschickt werden können</li>
</ul>
<p>Der Internetdienst "Photon", den Acastus-Photon zum auflösen von Adressen verwendet,
ist eine Alternative zur Google Maps API und anderen Geocoding Diensten,
welche auf einem eigenen Server betrieben werden kann (siehe https://github.com/komoot/photon).</p>
